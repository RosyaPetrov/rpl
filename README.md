# RPL - язык программирования для описания моделей

Данный проект позволяет ускорить разработку программного обеспечения на языке Go.
Он собирает всё описание модели в один файл, что избавляет разработчика от необходимости прописывать модель в нескольких местах и упрощает обновление данных.
